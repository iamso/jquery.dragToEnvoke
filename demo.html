Draggy Switch

Description:

A draggable UI switch element. Works in all browsers including Mobile Safari. Takes optional callbacks to fire when switching on/off, can be set to run once. Listens for a custom event to fire so it can be reset asynchronously.

Usage:

$('input[type=checkbox]')
	.dragToEnvoke({
		onFunc : function(el, parent, checkbox) { },
		offFunc : function(el, parent, checkbox) { },
		returnTime : 100,
		balancePoint : 0.5,
		runOnce : false,
		handleText : "",
		defaultState : '' //on or off
	});
	
//Trigger a state change
$('input[type=checkbox]').click();

See source for all options.
